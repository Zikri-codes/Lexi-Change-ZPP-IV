LEXI CHANGE

Lexi change is a web that converts text using American Standard Code for Information Interchange (ASCII) into other formats such as binary or hexadecimal

HOW TO USE IT?

  • Step 1: Type or paste your text into the input box
  • Step 2: Select your desired input / output format (binary,hexadecimal,etc.)
  • Step 3: Click the convert button
  • Step 4: View the converted result in the output box

Frequently Asked Question

F.A.Q

What kind of text can I convert?

expand_more

You can convert normal text like letters, numbers, and common symbols. Some special characters may give unexpected results.

Is there a text length limit?

expand_more

No stritch limit, but very long text might take more time to convert.

Can I convert back from Binary/Hex/Octal/ASCII to text?

expand_more

Yes! Just paste the converted result back into the input area and convert again as long as the input format is correct.

Can I change the separator between outputs?

expand_more

Yes, but for now it only supports a whitespace or no sparator. You can switch between these two options using the separator slider, and you can access it through the settings icon in the navigation bar for easy adjustments.

Does it update automatically as I type?

expand_more

For now. The converter doesn't update live as you type, so you’ll need to press the convert button to see the results.
