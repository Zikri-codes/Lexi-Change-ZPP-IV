## 📌 Description
A simple web-based **text converter** built with HTML, CSS, and JavaScript.  
Users can **convert text** between different formats or manipulate strings easily through the web.

## ✨ Features
- **Text transformations**:  
  - Binary
  - Hexadecimal
  - Octal
  - ASCII (numeric)
- **Simple and clean UI**.
